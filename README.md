# 简介
* 此文件包括：（1）答辩所用ppt（2）作品安装包（apk文件）
## 安装包使用说明：
* 下载.apk文件到手机
* 点击安装（若以前下载过本app可重新下载，内部记录不会丢失！）
* 启动app
注：本app支持Android5.0系统以上版本使用（就是几乎99％的Android手机和平板都能使用，可当做废话）
ios系统的用户无法使用，本人后续会持续更新ios等各种版本，敬请期待！
